**print(s)**: print string s to screen.

**+**: addition

**-**: subtract

*: multiply

**/**: divide

**%**: modulus

**<**: less than

**>**: greater than

**<=**: less than equal to

**>=**: greater than equal to

**f"{a}"**: format string with **a** injected into it.

**\n**: newline

**\t**: tab

**""""""**: multiline string

**"**: string literal

**with open(from_file) as f: [filestuff]**: do file operation with
    context manager.
